# 📱 Quiz MAUI App

A cross-platform quiz application built using **.NET MAUI**.  
This project demonstrates mobile application development with XAML-based UI, navigation using Shell, and structured page management.

---

## 🚀 Features

- 📋 Multiple quiz categories
- ❓ Interactive quiz interface
- 📊 Simple and clean UI using XAML
- 🔄 Navigation with AppShell
- ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

- **.NET MAUI**
- **C#**
- **XAML**
- **Visual Studio**

---

## 📁 Project Structure
QuizMauiApp/
│
├── Platforms/ # Platform-specific code (Android, iOS, Windows)
├── Resources/ # Images, fonts, styles
├── App.xaml # Global application resources
├── AppShell.xaml # Navigation structure
├── MainPage.xaml # Entry page
├── CategoriesPage.xaml # Categories UI
├── QuizPage.xaml # Quiz UI
├── MauiProgram.cs # App configuration
└── QuizMauiApp.csproj # Project file
